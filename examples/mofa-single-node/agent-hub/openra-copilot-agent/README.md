# OpenRA Copilot Agent

单节点 OpenRA 游戏控制 Agent，集成了 MCP 版本的所有工具功能。

## 功能特点

- ✅ **单节点设计**：所有游戏控制逻辑集中在一个 Agent 中
- ✅ **MCP 兼容**：使用与 MCP 版本完全相同的游戏 API 封装
- ✅ **智能命令解析**：支持自然语言命令，如 "生产 5 个步兵"
- ✅ **完整功能**：包含单位生产、移动、攻击、查询等所有功能
